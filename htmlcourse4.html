Now, let's learn more about TAGS!

With a tag for example the H2 tag: when you want to style it (we will learn that later) you can set a class. For example, <h2 class="class1">This is a heading</h2> will create a heading with the class "heading". You can then use CSS to style all elements with the class "heading".

Target them like this STYLE> .class1 { color: red; background-color: blue; font-family: Arial; font-size: 25px; font-weight: bold; width: 25px; height: 67px; border: none; border-radius: 3px; cursor: pointer; } /STYLE>

(The code may look weird with the heading, but it's correct!)
